# Portfolio Website

This project showcases a simple portfolio website with well-structured content and clean design. It highlights personal information, skills, and a contact form while following best practices in web development such as semantic HTML, external CSS styling, and responsive design

## Features
About, Skills, and Contact sections

Semantic HTML5 structure (header, nav, main, section, footer)

Internal navigation links

Styled contact form

Images with proper alt text

Hover effects and transitions

Responsive layout for mobile and desktop

## Technologies Used
HTML5

CSS3

Google Fonts
