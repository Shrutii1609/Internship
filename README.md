Portfolio Website
This project showcases a simple, interactive, and responsive portfolio website with well-structured content and a clean design. It presents personal information, skills, and a contact form while following best practices in web development such as semantic HTML, external CSS styling, JavaScript interactivity, and responsive design.

Features:
1.About, Skills, and Contact sections
2.Semantic HTML5 structure
(header, nav, main, section, footer)
3.Internal navigation links
4.Styled contact form with JavaScript validation
5.Images with proper alt text for accessibility
6.Hover effects, transitions, and animations
7.JavaScript interactivity (DOM manipulation & events)
8.Dark mode toggle with saved user preference
9.Responsive layout for mobile and desktop devices

Technologies Used
HTML5
CSS3
JavaScript (ES6)
Google Fonts
